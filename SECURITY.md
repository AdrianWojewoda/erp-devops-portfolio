# Security Policy

## Scope

This repository represents a DevOps portfolio project.
It is not a production SaaS system and does not process real user data.

## Reporting a Vulnerability

If you discover a security issue in the infrastructure configuration,
please open a GitHub Issue or contact the repository owner directly.

Sensitive information should not be disclosed publicly.

## Supported Versions

This project evolves incrementally and does not maintain long-term support versions.
Only the latest `main` branch state is considered active.

## Notes

- Secrets are not committed to the repository.
- ACME key material is excluded.
- Runtime credentials are stored outside version control.
